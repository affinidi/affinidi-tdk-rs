Key Finding
Affinidi TDK-RS is the most complete modern SSI stack in Rust. With the additions on the current branch (SD-JWT, SD-JWT-VC, mdoc, Status Lists, OpenID4VCI, OpenID4VP), it is now the only library covering all three pillars: secure messaging (DIDComm v2), credential formats (VC, SD-JWT-VC, mdoc), and exchange protocols (OID4VCI/VP) — while remaining fully platform-independent. The remaining gaps (BBS+, PQC, security audit) are additive enhancements, not architectural blockers.
